## Dasar Teori
Stack merupakan salah satu struktur data fundamental yang bekerja dengan prinsip LIFO (Last-In, First-Out), yaitu elemen yang terakhir dimasukkan adalah elemen pertama yang dikeluarkan. Operasi utama pada stack hanya dapat dilakukan pada bagian atas (top), dengan dua prosedur dasar yaitu push untuk menambah elemen dan pop untuk menghapus elemen dari puncak stack [1]. Stack dapat diimplementasikan menggunakan array maupun linked list. Implementasi berbasis array bersifat statis sehingga memiliki batasan ukuran dan dapat mengalami overflow ketika elemen yang dimasukkan melebihi kapasitas yang telah ditentukan [2]. Sebaliknya, implementasi berbasis linked list bersifat dinamis karena ukuran stack dapat berubah sesuai kebutuhan, sehingga lebih fleksibel dan mengurangi potensi overflow. Selain itu, operasi push dan pop pada stack berbasis linked list berjalan sangat efisien dengan kompleksitas waktu O(1) karena penanganannya selalu dilakukan pada node terdepan [1][3]. Secara keseluruhan, stack menjadi struktur penting dalam pemrograman karena dapat digunakan untuk berbagai kebutuhan seperti pengelolaan pemanggilan fungsi, proses undo/redo, hingga pengolahan ekspresi.

```C++
    }
}

            int count = 1;
            bool found = false;
            while(nodeBantu != Nil){
                if(count < posisi){
                    nodeBantu = nodeBantu->next;
                    count++;

                    cout << "Masukkan angka : ";

                    cout << "Data berhasil diupdate!" << endl;
                    cout << endl;
                    found = true;
                    break;
                }
            }
            if(found == false){
                cout << "Posisi " << posisi << " tidak valid!" << endl;
            }
        }
    }
}

    if(isEmpty(listStack) == true){
        cout << "List kosong!" << endl;
    } else {
```

```C++
#include "stack.h"
using namespace std;
```
```C++
// Fungsi untuk membuat stack baru dengan menginisialisasi top = Nil
void CreateStack(Stack &S) {
    S.top = Nil;  // Inisialisasi top dengan nilai Nil mendakan stack kosong
}

// Fungsi untuk mengecek apakah stack kosong
bool isEmpty(Stack S) {
    return S.top == Nil;  // Mengembalikan true jika top bernilai Nil, false jika tidak
}

// Fungsi untuk mengecek apakah stack penuh
bool isFull(Stack S) {
    return S.top == MaxEl - 1;  // Mengembalikan true jika top mencapai batas maksimum (MaxEl-1)
}

// Fungsi untuk menambah elemen ke dalam stack (push)
void push(Stack &S, infotype x) {
    if (!isFull(S)) {  // Cek apakah stack belum penuh
        S.top++;  // Increment nilai top
        S.info[S.top] = x;  // Masukkan nilai x ke array info pada indeks top
    } else {
        cout << "Stack Penuh!" << endl;  // Tampilkan pesan jika stack penuh
    }
}

// Fungsi untuk mengambil elemen teratas dari stack (pop)
infotype pop(Stack &S) {
    infotype x = -999;  // Inisialisasi nilai default jika stack kosong
    if (!isEmpty(S)) {  // Cek apakah stack tidak kosong
        x = S.info[S.top];  // Ambil nilai teratas
        S.top--;  // Decrement nilai top
    } else {
        cout << "Stack Kosong!" << endl;  // Tampilkan pesan jika stack kosong
    }
    return x;  // Kembalikan nilai yang diambil
}

// Fungsi untuk menampilkan isi stack
void printInfo(Stack S) {
    if (isEmpty(S)) {  // Cek apakah stack kosong
        cout << "Stack Kosong" << endl;  // Tampilkan pesan jika kosong
    } else {
        cout << "[TOP] ";  // Tampilkan penanda posisi top
        for (int i = S.top; i >= 0; i--) {  // Loop dari top sampai dasar stack
            cout << S.info[i] << " ";  // Tampilkan setiap elemen
        }
        cout << endl;  // Pindah baris setelah selesai
    }
}

// Fungsi untuk membalik urutan elemen dalam stack
void balikStack(Stack &S) {
    if (!isEmpty(S)) {  // Cek apakah stack tidak kosong
        Stack temp1, temp2;  // Deklarasi dua stack temporary
        CreateStack(temp1); CreateStack(temp2);  // Inisialisasi kedua stack temporary

        while (!isEmpty(S)) { push(temp1, pop(S)); }  // Pindahkan semua elemen ke temp1 (urutan terbalik)

        while (!isEmpty(temp1)) { push(temp2, pop(temp1)); }  // Pindahkan ke temp2 (urutan terbalik lagi)

        while (!isEmpty(temp2)) { push(S, pop(temp2)); }  // Kembalikan ke stack asal (urutan sudah benar)
    }
}
```

## Kesimpulan
Praktikum ini menunjukkan bahwa stack merupakan struktur data dasar yang bekerja dengan prinsip data yang terakhir masuk akan menjadi data yang pertama keluar, sehingga proses penambahan dan penghapusan hanya boleh dilakukan pada bagian puncak stack. Stack dapat dibuat menggunakan array yang bersifat statis dan memiliki batas kapasitas, atau menggunakan linked list yang bersifat dinamis dan lebih fleksibel karena tidak memiliki risiko kehabisan ruang. Operasi dasar seperti push dan pop pada versi linked list berjalan sangat efisien karena selalu dilakukan pada satu titik saja. Selain itu, praktikum ini juga membuktikan bahwa stack bisa dikembangkan dengan fitur tambahan seperti membalik urutan elemen, menyisipkan data agar tetap terurut naik, serta membaca input satu per satu menggunakan stack sementara sebagai media pemrosesan. Hal ini menunjukkan bahwa stack adalah struktur data yang sederhana namun kuat dan mudah diterapkan dalam berbagai kebutuhan pemrograman.
